# 🚀 快速启动指南

## 第一步：启动开发服务器

在项目根目录运行：

```bash
npm run dev
```

服务器将在 http://localhost:3000 启动

## 第二步：浏览页面

### 🏠 首页/仪表盘 (/)
- 查看数据概览（本月分析、生成文章、已发布、总阅读）
- 快速操作入口
- 最近文章列表

### 🔍 选题分析 (/analysis)
1. 在搜索框输入关键词（如"人工智能"、"ChatGPT"）
2. 点击"开始分析"按钮
3. 查看搜索到的文章列表
4. 点击"生成洞察报告"
5. 查看：
   - 点赞量 TOP 5
   - 互动率 TOP 5
   - 高频词云
   - 5个选题洞察

### ✍️ 内容创作 (/creation)
1. 选择创作方式：
   - 基于选题洞察（从分析结果中选择）
   - 自定义主题（输入自己的主题）
2. 选择内容风格、文章长度、配图数量
3. 点击"开始创作"
4. 等待 AI 生成（会显示进度）
5. 查看文章预览
6. 可以编辑或提交到发布管理

### 📝 发布管理 (/publish)
1. 查看所有文章统计
2. 使用搜索和筛选功能
3. 点击文章卡片上的操作按钮：
   - 草稿：编辑、预览、发布
   - 已发布：查看数据、编辑、在公众号查看
   - 发布失败：编辑、重新发布
4. 支持批量操作（多选后显示批量操作按钮）

## 🎨 自定义样式

### 修改主题色
编辑 `app/globals.css` 中的 CSS 变量：

```css
:root {
  --primary: 221.2 83.2% 53.3%;  /* 主色调 */
  --secondary: 210 40% 96.1%;     /* 次要色 */
  /* 更多颜色变量... */
}
```

### 修改布局
编辑 `app/layout.tsx` 来调整整体布局

### 添加新页面
1. 在 `app/` 目录下创建新文件夹
2. 添加 `page.tsx` 文件
3. 在 `components/sidebar.tsx` 中添加导航链接

## 🐛 常见问题

### Q: 端口被占用怎么办？
A: 修改端口：`npm run dev -- -p 3001`

### Q: 样式没有生效？
A: 确保 Tailwind CSS 配置正确，重启开发服务器

### Q: TypeScript 报错？
A: 运行 `npm run lint` 查看详细错误
